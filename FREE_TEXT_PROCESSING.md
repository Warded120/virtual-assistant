# Обробка вільного тексту (Free Text Processing)

## Огляд

Віртуальний асистент тепер підтримує обробку природного тексту без необхідності використання команд. Користувачі можуть писати звичайними фразами українською або англійською мовами, і бот автоматично визначить намір (intent) та маршрутизує запит до відповідного обробника.

## Архітектура

### Компоненти

1. **TextAnalyzerService** - аналізує текст та визначає намір користувача
2. **FreeTextHandler** - обробляє вільний текст та маршрутизує до відповідних команд
3. **TelegramBot** - оновлений для підтримки як команд, так і вільного тексту

### Потік обробки

```
Користувач надсилає повідомлення
    ↓
TelegramBot отримує повідомлення
    ↓
Перевірка: чи починається з "/"?
    ↓ (Так)           ↓ (Ні)
CommandHandler    FreeTextHandler
    ↓                  ↓
                TextAnalyzerService
                       ↓
                Визначення наміру
                       ↓
                CommandHandler
                       ↓
Відповідь користувачу
```

## Підтримувані наміри (Intents)

### 1. Погода (Weather)
**Ключові слова:**
- Українська: погода, температура, дощ, сонце, хмарно, холодно, тепло
- English: weather, temperature, rain, sun, cloudy, cold, warm

**Приклади запитів:**
- "Яка сьогодні погода?"
- "What's the weather like?"
- "Холодно чи тепло на вулиці?"
- "Will it rain today?"

### 2. Валюта (Currency)
**Ключові слова:**
- Українська: валюта, курс, долар, євро, гривня, обмін
- English: currency, exchange, dollar, euro, USD, EUR, UAH

**Приклади запитів:**
- "Який курс долара?"
- "What's the exchange rate?"
- "Скільки коштує євро?"
- "USD to UAH"

### 3. Новини (News)
**Ключові слова:**
- Українська: новини, новость, стаття, події, що нового, що сталося
- English: news, article, events, what's new, happened

**Приклади запитів:**
- "Які останні новини?"
- "What's new today?"
- "Що сталося сьогодні?"
- "Show me the news"

## Додаткові можливості

### Витягування сутностей (Entity Extraction)

#### Локація
Система може витягати назви міст з тексту:
- "Яка погода **в Києві**?" → Витягує: "Києві"
- "Weather **in London**" → Витягує: "London"

#### Валюта
Система розпізнає коди валют:
- "Курс **USD**" → Витягує: "USD"
- "Скільки коштує **євро**?" → Витягує: "EUR"

### Рівень впевненості (Confidence Score)

Кожен аналіз тексту повертає рівень впевненості (0.0 - 1.0), який базується на кількості збігів ключових слів.

## Розширення функціональності

### Додавання нових намірів

Щоб додати новий намір:

1. Додайте ключові слова в `TextAnalyzerService.INTENT_KEYWORDS`:
```java
INTENT_KEYWORDS.put("новий_намір", Arrays.asList(
    "ключове_слово1", "keyword1", "ключове_слово2"
));
```

2. Додайте маппінг в `FreeTextHandler.mapIntentToCommand()`:
```java
case "новий_намір" -> "/нова_команда";
```

3. Створіть відповідний CommandHandler для нової команди

### Додавання витягування сутностей

Додайте новий метод у `TextAnalyzerService`:
```java
public String extractNewEntity(String text) {
    // Логіка витягування
    return extractedEntity;
}
```

## Переваги підходу

✅ **Природна взаємодія** - користувачі можуть писати звичайною мовою
✅ **Багатомовність** - підтримка української та англійської
✅ **Розширюваність** - легко додавати нові наміри та ключові слова
✅ **Відмовостійкість** - якщо намір не визначено, показується допомога
✅ **Логування** - всі етапи аналізу логуються для налагодження

## Майбутні покращення

- Інтеграція з OpenNLP або Stanford NLP для покращеного аналізу
- Підтримка контексту розмови (conversation context)
- Машинне навчання для покращення визначення намірів
- Підтримка додаткових мов
- Витягування складних сутностей (дати, часу, чисел)
